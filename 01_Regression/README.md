# 📈 Regression

This module covers regression algorithms — predicting continuous target values from input features. Each notebook builds on the last, moving from simple linear models to regularized and more robust variants.

Part of the [**Portfolio Projects - Machine Learning**](../) repository.

<br/>

## 📂 Contents

| # | Notebook | Concept Covered |
|---|---|---|
| 1 | `Placement-Project-Logistic Regression` | Single-feature regression, line of best fit, OLS |
| 2 | `02_multiple_linear_regression.ipynb` | Multi-feature regression, coefficients & interpretation |
| 3 | `03_polynomial_regression.ipynb` | Non-linear relationships via polynomial features |
| 4 | `04_gradient_descent.ipynb` | Cost function optimization, learning rate tuning |
| 5 | `05_regularization_ridge_lasso.ipynb` | Ridge (L2), Lasso (L1), bias-variance tradeoff |

<br/>

## 🧠 Key Concepts

- **Assumptions of Linear Regression** — linearity, independence, homoscedasticity, normality of residuals
- **Cost Function** — Mean Squared Error minimization
- **Gradient Descent** — batch, stochastic, and mini-batch variants
- **Bias-Variance Tradeoff** — why regularization matters
- **Regularization** — Ridge (L2 penalty) vs Lasso (L1 penalty, feature selection)
- **Model Evaluation** — choosing the right metric for the problem
